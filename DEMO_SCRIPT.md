# 2-3 Minute Demo Script

## 0:00-0:20 - Intro

- "This is Fortune Spin Arena, a Wheel-of-Fortune-style game built with AI-assisted development."
- "It includes a playable frontend and a backend with persistent sessions and leaderboard."

## 0:20-1:00 - Basic gameplay (Part 1)

- Create a new session with your name.
- Spin the wheel.
- Guess a consonant.
- Buy a vowel.
- Attempt solve (optional wrong attempt to show penalty).
- Solve correctly and show final score.

## 1:00-1:45 - System expansion (Part 2)

- Show session ID and explain resume capability.
- Refresh page and resume using saved/entered session ID.
- Open leaderboard and show persisted completed score.
- Show timeline with recorded actions.

## 1:45-2:20 - Replay and backend-driven data

- Click replay to start a new round linked to previous session.
- Mention wheel/puzzle configuration is API-driven from backend.
- Mention all state changes are synchronized through API responses.

## 2:20-2:45 - Wrap-up

- Summarize: "Playable game + persistent scoring + session tracking + resume/replay delivered."
- Mention next-phase extension ideas: multiplayer turns, DB migration, auth, difficulty balancing.
